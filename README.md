## Installation

AwesomeMedia is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "AwesomeMedia", git: 'https://github.com/iOSWizards/AwesomeMedia.git', tag: '0.2.4'
```
